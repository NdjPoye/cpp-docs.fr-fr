# <a name="strlwr-wcslwr"></a>strlwr, wcslwr

Ces fonctions POSIX sont déconseillées. Utilisez plutôt les fonctions [_strlwr, _wcslwr, _mbslwr, _strlwr_l, _wcslwr_l, _mbslwr_l](strlwr-wcslwr-mbslwr-strlwr-l-wcslwr-l-mbslwr-l.md), qui sont conformes à la norme ISO C++, ou [_strlwr_s, _strlwr_s_l, _mbslwr_s, _mbslwr_s_l, _wcslwr_s, _wcslwr_s_l](strlwr-s-strlwr-s-l-mbslwr-s-mbslwr-s-l-wcslwr-s-wcslwr-s-l.md), qui offrent une sécurité améliorée.
