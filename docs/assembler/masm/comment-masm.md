# <a name="comment-masm"></a>COMMENT (MASM)
Traite tous les *texte* entre ou sur la même ligne que les délimiteurs comme un commentaire.  
  
## <a name="syntax"></a>Syntaxe  
  
```  
  
      COMMENT delimiter [[text]]   
[[text]]   
[[text]] delimiter [[text]]  
```
