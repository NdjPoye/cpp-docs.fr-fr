# <a name="operator-high32"></a>HIGH32, opérateur
Retourne les 32 bits de poids fort de *expression*. Expressions MASM sont des valeurs 64 bits.  
  
## <a name="syntax"></a>Syntaxe  
  
```  
  
HIGH32 expression  
```
