# <a name="operator-low32"></a>LOW32, opérateur
Retourne les 32 bits de poids faibles de *expression*. Expressions MASM sont des valeurs 64 bits.  
  
## <a name="syntax"></a>Syntaxe  
  
```  
  
LOW32 expression  
```
