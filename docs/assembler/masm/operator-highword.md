# <a name="operator-highword"></a>HIGHWORD, opérateur
Retourne les 16 bits de poids fort de 32 bits de poids faibles de *expression*. Expressions MASM sont des valeurs 64 bits.  
  
## <a name="syntax"></a>Syntaxe  
  
```  
  
HIGHWORD expression  
```
