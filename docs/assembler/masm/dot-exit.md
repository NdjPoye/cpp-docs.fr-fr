# <a name="exit"></a>.EXIT
Génère le code d’arrêt.  
  
## <a name="syntax"></a>Syntaxe  
  
```  
  
.EXIT [[expression]]  
```  
  
## <a name="remarks"></a>Notes  
 Retourne facultatif *expression* au shell.
