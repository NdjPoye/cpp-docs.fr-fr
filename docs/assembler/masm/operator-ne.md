# <a name="operator-ne"></a>NE, opérateur
Retourne la valeur true (-1) si *expression1* n’est pas égale *expression2*, ou la valeur false (0) dans ce cas.  
  
## <a name="syntax"></a>Syntaxe  
  
```  
  
expression1 NE expression2  
```
