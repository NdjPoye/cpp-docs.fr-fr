# <a name="while-masm"></a>WHILE (MASM)
Répète l’ensemble du bloc `statements` tant que *expression* conserve la valeur true.  
  
## <a name="syntax"></a>Syntaxe  
  
```  
  
   WHILE expression  
statements  
ENDM  
```
