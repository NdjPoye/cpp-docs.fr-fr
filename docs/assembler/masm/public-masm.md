# <a name="public-masm"></a>PUBLIC (MASM)
Permet de chaque variable, une étiquette ou un symbole absolu spécifié en tant que *nom* disponibles pour tous les autres modules dans le programme.  
  
## <a name="syntax"></a>Syntaxe  
  
```  
  
PUBLIC [[langtype]] name [[, [[langtype]] name]]...  
```
