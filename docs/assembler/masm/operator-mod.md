# <a name="operator-mod"></a>MOD, opérateur
Retourne la valeur entière du reste (modulo) lors de la division *expression1* par *expression2*.  
  
## <a name="syntax"></a>Syntaxe  
  
```  
  
expression1 MOD expression2  
```
