# <a name="qword"></a>QWORD
Alloue et initialise éventuellement de 8 octets pour chaque `initializer`. Peut également servir comme un spécificateur de type n’importe où un type est autorisé.  
  
## <a name="syntax"></a>Syntaxe  
  
```  
[[name]] QWORD initializer [[, initializer]]...  
```
