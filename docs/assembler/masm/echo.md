# ECHO
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

affiche le *message au* périphérique de sortie standard \(par défaut, l'écran\).  
  
## Syntaxe  
  
```  
  
ECHO   
message  
  
```  
  
## Notes  
 mêmes que [%OUT](../../assembler/masm/percent-out.md).
