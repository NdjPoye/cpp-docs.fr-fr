# <a name="operator-low"></a>LOW, opérateur
Retourne les 8 bits de poids faibles de *expression*. Expressions MASM sont des valeurs 64 bits.  
  
## <a name="syntax"></a>Syntaxe  
  
```  
  
LOW expression  
```
