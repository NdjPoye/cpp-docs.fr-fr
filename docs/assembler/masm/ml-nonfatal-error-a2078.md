# <a name="ml-nonfatal-error-a2078"></a>Erreur ML non fatale A2078
**instruction n’autorise pas FAR adressage indirect**  
  
 Une boucle ou renvoi conditionnel ne peut pas prendre un opérande de mémoire. Il doit comporter une adresse relative ou une étiquette.
