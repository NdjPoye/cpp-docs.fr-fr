# <a name="operator-amp"></a>(opérateur)&amp;
Au niveau du bit **AND**. Utilisé uniquement dans [. IF](../../assembler/masm/dot-if.md), [. Alors que](../../assembler/masm/dot-while.md), ou [. RÉPÉTEZ](../../assembler/masm/dot-repeat.md) bloque et évaluées au moment de l’exécution, pas au moment de l’assembly.  
  
## <a name="syntax"></a>Syntaxe  
  
```  
  
expression1 & expression2  
```
