# operator THIS
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Retourne un opérande d' `type` spécifié dont les valeurs d'offset et de segment est égale à la valeur de compteur de position actuelle.  
  
## Syntaxe  
  
```  
  
THIS type  
```
