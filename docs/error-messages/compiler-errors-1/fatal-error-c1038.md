# Erreur irr&#233;cup&#233;rable C1038
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

limite du compilateur : fonction : état du flux de contrôle trop complexe ; simplifiez la fonction  
  
 La fonction a plus d’états de flux de contrôle que le compilateur ne peut gérer. Simplifiez le flux de contrôle ou fractionnez la fonction en fonctions plus petites.
