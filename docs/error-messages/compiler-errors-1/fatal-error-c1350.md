# <a name="fatal-error-c1350"></a>Erreur irrécupérable C1350
Erreur lors du chargement de la dll 'dll' : dll introuvable  
  
 La DLL qui prend en charge l’opération tentée est introuvable.  
  
 Cela indique un problème avec votre installation et vous devez réinstaller la DLL à partir du CD-ROM du produit.
