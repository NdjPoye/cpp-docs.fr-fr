# Erreur du compilateur C2151
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

plus d'un attribut de langage  
  
 Une fonction contient plus d'un mot clé \(`__cdecl`, `__stdcall` ou `__fastcall`\) spécifiant une convention d'appel.
