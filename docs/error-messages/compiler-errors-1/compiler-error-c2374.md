# <a name="compiler-error-c2374"></a>Erreur du compilateur C2374
'identificateur' : redéfinition ; initialisation multiple  
  
 L’identificateur est initialisé plusieurs fois.  
  
 L’exemple suivant génère l’erreur C2374 :  
  
```  
// C2374.cpp  
// compile with: /c  
int i = 0;  
int i = 1;   // C2374  
int j = 1;   // OK  
```
