# Erreur du compilateur C2193
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'identificateur' : déjà dans un segment  
  
 Une fonction a été placée dans deux segments différents en utilisant les pragmas `alloc_text` et `code_seg`.  
  
 L'exemple suivant génère l'erreur C2193 :  
  
```  
// C2193.cpp  
// compile with: /c  
extern "C" void MYFUNCTION();  
#pragma alloc_text(MYCODE, MYFUNCTION)  
#pragma code_seg("MYCODE2")  
extern "C" void MYFUNCTION() {}   // C2193  
extern "C" void MYFUNCTION2() {}  
```
