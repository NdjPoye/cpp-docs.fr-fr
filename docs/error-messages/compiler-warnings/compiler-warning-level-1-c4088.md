# <a name="compiler-warning-level-1-c4088"></a>Compilateur avertissement (niveau 1) C4088
'fonction' : incompatibilité de pointeur dans le paramètre actuel 'nombre', paramètre formel 'nombre'  
  
 Les paramètres formels et réels correspondants ont un niveau d’indirection différent. Le paramètre réel est transmis sans modification. La fonction appelée interprète sa valeur comme un pointeur.
