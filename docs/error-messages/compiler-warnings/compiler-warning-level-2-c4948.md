# <a name="compiler-warning-level-2-c4948"></a>Avertissement du compilateur (niveau 2) C4948
type de retour de 'accesseur' ne correspond pas au dernier type de paramètre de la méthode setter correspondante  
  
Le compilateur a détecté une incompatibilité entre le type de données est en cours d’obtenir et définie pour une propriété indexée.  
  
C4948 est uniquement accessible à l’aide de l’option du compilateur obsolète **oldSyntax ;**.
