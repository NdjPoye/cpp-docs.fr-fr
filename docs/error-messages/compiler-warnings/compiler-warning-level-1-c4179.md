# <a name="compiler-warning-level-1-c4179"></a>Avertissement du compilateur (niveau 1) C4179
' / / *' : analysé comme '/' et ' /\*' : confusion avec standard ' / /' commentaires  
  
 **//\*** est un délimiteur de commentaire incorrect. Utilisez plutôt **//** ou **/\*** .
