# <a name="compiler-warning-level-1-c4379"></a>Avertissement du compilateur (niveau 1) C4379
Une version du common language runtime n’est pas prise en charge par ce compilateur. À l’aide de cette version peut entraîner des résultats inattendus.  
  
 Vous avez une version précédente du common language runtime sur votre ordinateur, mais pas la version actuelle. Pour résoudre C4379, installez la version du common language runtime fourni avec votre compilateur.
