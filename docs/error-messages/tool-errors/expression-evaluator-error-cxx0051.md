# <a name="expression-evaluator-error-cxx0051"></a>Évaluateur d'expression, erreur CXX0051
plusieurs symboles surchargés sont spécifiés au point d’arrêt  
  
 L’évaluateur d’expression n’a pas pu déterminer parmi plusieurs symboles surchargés à utiliser comme point d’arrêt.  
  
 Cette erreur est identique à CAN0051.
