# Erreur math&#233;matique M6107
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

non émulé  
  
 Une tentative a eu lieu pour exécuter une instruction de coprocesseur non valide ou qui n'est pas prise en charge par l'émulateur.  
  
 Le programme se termine par le code de sortie 135.
