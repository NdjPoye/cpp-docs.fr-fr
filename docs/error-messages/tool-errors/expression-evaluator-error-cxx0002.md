# <a name="expression-evaluator-error-cxx0002"></a>Évaluateur d'expression, erreur CXX0002
accès à la mémoire utilisateur erreur  
  
 L’expression tente de la référence de la mémoire qui n’est pas allouée au programme en cours de débogage.  
  
 Cette erreur est identique à CAN0002.
