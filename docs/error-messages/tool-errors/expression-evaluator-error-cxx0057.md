# <a name="expression-evaluator-error-cxx0057"></a>Évaluateur d'expression, erreur CXX0057
Aucun code au numéro de ligne  
  
 Aucun code n’a été généré pour le numéro de ligne spécifié. Il ne peut pas être utilisé comme point d’arrêt.  
  
 Cette erreur est identique à CAN0057.
