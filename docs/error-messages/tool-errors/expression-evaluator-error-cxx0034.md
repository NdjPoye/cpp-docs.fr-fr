# <a name="expression-evaluator-error-cxx0034"></a>Évaluateur d'expression, erreur CXX0034
**types incompatibles avec l’opérateur**  
  
 Les types d’opérande spécifiés ne sont pas autorisés pour l’opération.  
  
 Par exemple, un pointeur ne peut pas être multiplié par n’importe quelle valeur.  
  
 Vous devrez peut-être taper convertir les opérandes dans un type compatible avec l’opérateur.
