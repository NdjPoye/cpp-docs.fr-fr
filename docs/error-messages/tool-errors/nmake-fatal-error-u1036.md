# <a name="nmake-fatal-error-u1036"></a>Erreur irrécupérable NMAKE U1036
Erreur de syntaxe : noms trop nombreux à gauche de '='  
  
 Chaîne qu’une seule est autorisée à gauche d’une définition de macro.
