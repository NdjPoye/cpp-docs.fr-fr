# <a name="expression-evaluator-error-cxx0047"></a>Évaluateur d'expression, erreur CXX0047
liste d’arguments ne correspond pas à une fonction  
  
 Une expression a appelé une fonction avec une liste de paramètres réels qui ne correspond pas à la liste de paramètres formels d’une fonction portant le même nom que celui défini dans le programme.  
  
 Les fonctions surchargées peuvent être appelées uniquement s’il existe une correspondance exacte de paramètre ou une correspondance qui ne requiert pas de la construction d’un objet.  
  
 Cette erreur est identique à CAN0047.
