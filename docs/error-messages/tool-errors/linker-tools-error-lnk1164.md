# <a name="linker-tools-error-lnk1164"></a>Erreur des outils Éditeur de liens LNK1164
section alignement de section (nombre) supérieur à la valeur /ALIGN  
  
 La taille d’alignement pour la section donnée dans le fichier objet dépasse la valeur spécifiée avec la [/ALIGN](../../build/reference/align-section-alignment.md) option. Le **/ALIGN** valeur doit être une puissance de 2 égale ou supérieure à l’alignement de section donné dans le fichier objet.  
  
 Soit recompiler avec un petit alignement de section ou augmentez la **/ALIGN** valeur.
