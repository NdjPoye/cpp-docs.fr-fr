# <a name="expression-evaluator-error-cxx0044"></a>Évaluateur d'expression, erreur CXX0044
utilisation du pointeur de _based (void) requiert l’opérateur : >  
  
 Un pointeur basé sur `void` ne peut pas être utilisée directement. Vous devez constituer un pointeur complet à l’aide de la **: >** opérateur.  
  
 Cette erreur est identique à CAN0044.
