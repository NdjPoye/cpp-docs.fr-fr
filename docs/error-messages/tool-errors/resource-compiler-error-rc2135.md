# <a name="resource-compiler-error-rc2135"></a>Erreur RC2135 du compilateur de ressources 
fichier introuvable : nom de fichier  
  
 Le fichier spécifié sur la ligne de commande du compilateur de ressources est introuvable. Vérifiez que le fichier a été déplacé vers un autre répertoire et que le nom de fichier et le chemin sont corrects.  
  
 Les fichiers sont recherchés à l’aide du **INCLUDE** variable d’environnement ou de Visual C++ **INCLUDE** paramètre.
