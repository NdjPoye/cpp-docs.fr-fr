# <a name="resource-compiler-fatal-error-rc1101"></a>Erreur irrécupérable RC1101 du compilateur de ressources 
Aucun nom de fichier binaire ressource spécifiée  
  
 La sortie de renommer (/ fo) option n’est pas suivie par un nom de fichier.  
  
 Utilisez la syntaxe suivante pour l’option /fo :  
  
 **/Fo RC**   
 ***nomfichier_sortie nomfichier_entrée.*  rc**
