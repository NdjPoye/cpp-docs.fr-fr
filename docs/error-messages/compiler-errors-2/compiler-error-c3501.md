# <a name="compiler-error-c3501"></a>Erreur du compilateur C3501
aucune typelib inscrite pour ProgID 'progid'  
  
 L’ID de classe pour un progid donné ne possède pas de bibliothèque de type associée. Par conséquent, vous ne pouvez pas passer ce progid à la `#import` instruction.
