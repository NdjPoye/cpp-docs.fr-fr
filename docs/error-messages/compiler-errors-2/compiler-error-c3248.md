# <a name="compiler-error-c3248"></a>Erreur du compilateur C3248
'fonction1' : fonction déclarée comme '__sealed' ne peut pas être substituée par 'fonction2'  
  
Une classe dérivée a tenté de substituer une méthode virtuelle **__sealed** .  
  
C3248 est uniquement accessible à l’aide de l’option du compilateur obsolète **oldSyntax ;**.
