# Erreur du compilateur C2587
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'identificateur' : utilisation non conforme d'une variable locale comme paramètre par défaut  
  
 Les variables locales ne sont pas autorisées comme paramètres par défaut.  
  
 L'exemple suivant génère l'erreur C2587 :  
  
```  
// C2587.cpp  
// compile with: /c  
int i;  
void func() {  
   int j;  
   extern void func2( int k = j );  // C2587 -- local variable  
   extern void func3( int k = i );   // OK  
}  
```
