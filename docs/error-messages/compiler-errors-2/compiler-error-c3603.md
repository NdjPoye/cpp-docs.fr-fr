# <a name="compiler-error-c3603"></a>Erreur du compilateur C3603
'Symbole' : type 'Type' non encore pris en charge.  
  
 Vous avez tenté d’utiliser un type de données qui n’est pas encore pris en charge par le runtime .NET dans un objet managé.
