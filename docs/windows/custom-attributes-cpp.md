# <a name="custom-attributes-c"></a>Attributs personnalisés (C++)
Un attribut personnalisé est une technique fortement typée qui permet à l’utilisateur d’étendre les métadonnées. Cette rubrique porte sur l’utilisation d’attributs : spécification des paramètres et des cibles.  
  
 Pour plus d’informations sur la définition de nouveaux attributs personnalisés, consultez,
