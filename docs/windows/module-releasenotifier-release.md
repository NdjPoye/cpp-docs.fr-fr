# <a name="modulereleasenotifierrelease"></a>Module::ReleaseNotifier::Release
Supprime l’objet Module::ReleaseNotifier actuel si l’objet a été construit avec un paramètre de `true`.  
  
## <a name="syntax"></a>Syntaxe  
  
```cpp  
void Release() throw();  
```
