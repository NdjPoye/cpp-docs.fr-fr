# CCommonDialog Class
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

La classe de base des classes qui encapsulent des fonctionnalités du communs windows dialogue.  
  
## Syntaxe  
  
```  
class CCommonDialog : public CDialog  
```
