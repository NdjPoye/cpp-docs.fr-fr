# Classes internes
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Les classes suivantes sont utilises en interne dans MFC.  Par souci d'exhaustivit, cette section dcrit ces classes internes, mais elles ne sont pas destines  tre utilises directement dans votre code.  
  
## Dans cette section  
  
|Classe|Description|  
|------------|-----------------|  
|`CCommandManager Class`|La classe `CCommandManager` gre les commandes et leur association avec des images.|  
|`CDialogImpl Class (MFC)`|Fournit les dtails d'implmentation des classes bases sur une bote de dialogue, telles que `CDialogEx`.|  
|`CDocumentAdapter Class`|Implmente l'interface IDocument requise pour la recherche et l'organisation de responsables.|  
|`CFrameImpl Class`|La classe `CFrameImpl` gre la personnalisation de la barre d'outils pour les classes de cadre de fentresuivantes: [CFrameWndEx Class](../../mfc/reference/cframewndex-class.md), [Classe CMDIFrameWndEx](../../mfc/reference/cmdiframewndex-class.md), et [COleIPFrameWndEx Class](../../mfc/reference/coleipframewndex-class.md).|  
|`CFullScreenImpl Class`|Implmente des fonctionnalits de plein cran communes aux applications SDI et MDI.|  
|`CMDIClientAreaWnd Class`|La classe `CMDIClientAreaWnd` est une classe d'assistance qui simplifie l'implmentation des onglets MDI et des groupes tabuls MDI.|
