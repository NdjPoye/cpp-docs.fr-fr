# <a name="endcolumnmap"></a>END_COLUMN_MAP
Marque la fin des entrées de mappage de colonne.  
  
## <a name="syntax"></a>Syntaxe  
  
```cpp
END_COLUMN_MAP()  
  
```  
  
## <a name="remarks"></a>Notes  
 Il est utilisé avec un seul accesseur sur un ensemble de lignes. La macro `BEGIN_COLUMN_MAP` se termine avec la macro `END_COLUMN_MAP` .  
  
## <a name="example"></a>Exemple  
 Consultez [BEGIN_COLUMN_MAP](../../data/oledb/begin-column-map.md).
