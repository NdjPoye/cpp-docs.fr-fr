# MSBuild Error MSB3171
**MSB3171 : Problème lors de la génération du manifeste. '  \<erreur\>'**  
  
 Ceci est une erreur inattendue générée lors de l'obtention d'informations sur le fichier ou l'assembly référencé.  Ce message peut être généré lors de la génération du manifeste d'application ou de déploiement.  
  
### Pour corriger cette erreur  
  
-   Pour plus d'informations sur la cause d'une erreur MSBuild et les résolutions possibles, consultez [Additional Resources](../Topic/Additional%20MSBuild%20Resources.md).
